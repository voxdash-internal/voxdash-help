title: "Editor Panel – Designer Tab"
---

After creating a questionnaire—either manually or by importing a supported file—you are automatically redirected to the **Editor Panel**.

The **Designer** tab is the primary workspace where you build, edit, and organize your questionnaire. From this page, you can add questions, configure questionnaire settings, define survey behavior, manage quotas, estimate interview length, generate Text-to-Speech (TTS) audio, and much more.

Whether you are designing a simple customer feedback form or a complex multilingual market research survey, the Designer tab provides all the tools required to create and manage your questionnaire.

## Designer Tab Layout

The Designer tab consists of four main areas:

* **Top Toolbar** – Access questionnaire overview, team management, quota management, and search tools.
* **Question Library (Left Panel)** – Add new questions to your questionnaire.
* **Questionnaire Canvas (Center)** – Build and organize questionnaire pages and questions.
* **Properties Panel (Right Panel)** – Configure questionnaire settings or edit the selected question.

Each area is described in the following sections.

## Top Toolbar

The toolbar at the top of the Designer tab provides quick access to commonly used tools and questionnaire management features.

### Questionnaire Overview

The **Overview** panel provides a high-level summary of your questionnaire.

It displays key statistics, including:

* Total number of questions
* Question types used
* Number of tagged questions
* Number of questions containing logic
* Other questionnaire metrics

Each statistic is displayed as an interactive card.

Clicking a card automatically filters the questionnaire to show only the corresponding questions.

For example, you can quickly display:

* All Radio Button questions
* All Demographic questions
* Questions containing logic
* Questions with validation rules

The Overview panel is particularly useful when working with large questionnaires.

### Team

The **Team** section allows you to collaborate with other users by granting access to the questionnaire.

Depending on user permissions, team members may be able to:

* View questionnaires
* Edit questions
* Review questionnaire content
* Participate in questionnaire development

> **For detailed information, refer to the Team Management documentation.**

### Quota

The **Quota** button opens the **Quota Management** page.

From here you can:

* Create quota groups
* Edit existing quota groups
* Validate quota configurations
* Import and export quotas

Only questions tagged as **Quota** are available for quota configuration.

> **For more information, refer to the Quota Management documentation.**

## Search and Filter

The search tools help you quickly locate questions within large questionnaires.

Available search options include:

### Search

Search for questions by:

* Question ID
* Question title

### Sections

Display questions assigned to a specific **Section Tag**.

### Mode

Filter questions by their assigned **Data Collection Mode**.

### Language

Display questions tagged for a selected language using the **Multi-Language** tag.

### Screener

Display only questions marked as **Screener** questions.

> **For detailed information, refer to the Search and Filter documentation.**

## Question Library

The left panel contains the **Question Library**, which includes every available question type.

From this panel you can add questions such as:

* Radio Button
* Checkboxes
* Dropdown
* Rating Scale
* Matrix
* Dynamic Panel
* HTML
* Image
* Signature
* File Upload

Simply drag a question into the questionnaire or select it to add it to the current page.

> **For a complete description of each question type, refer to the Available Question Types documentation.**

## Properties Panel

The panel on the right side changes depending on what is currently selected.

* If **no question is selected**, the panel displays **Questionnaire Settings**.
* If **a question is selected**, the panel displays **Question Settings** for that specific question.

### Questionnaire Settings

Questionnaire Settings control the overall behavior and appearance of the survey.

#### General Settings

The **General Settings** section contains the basic questionnaire properties.

Available options include:

**Questionnaire Title**

The main title displayed at the beginning of the questionnaire.

**Questionnaire Description**

An optional description that introduces the survey or provides additional instructions.

**Show Title and Description**

Choose whether the questionnaire title and description are visible to respondents.

**Read-Only Questionnaire**

Makes the questionnaire read-only, preventing respondents from modifying answers.

This option is useful for previewing or displaying completed questionnaires.

**Questionnaire Language**

Select the language currently displayed in the editor.

For multilingual questionnaires, changing the language allows you to edit and preview questions in the selected language.

**Limit to One Response**

Restricts respondents to a single questionnaire submission.

**Questionnaire Width Mode**

Choose how the questionnaire is displayed on the screen.

Available options:

* **Auto** – Width is determined automatically.
* **Static** – Uses a fixed width.
* **Responsive** – Automatically adjusts to different screen sizes.

#### Length of Interview (LOI)

The **Length of Interview (LOI)** feature estimates how long respondents are expected to take to complete the questionnaire.

The estimate assists with:

* Project planning
* Respondent experience
* Budget estimation
* Incentive planning
* Service tier selection

LOI estimates are based on:

* Question types
* Data collection method
* Configurable timing multipliers

> **For detailed configuration instructions, refer to the LOI documentation.**

#### Text-to-Speech (TTS)

The **Text-to-Speech (TTS)** feature converts questionnaire content into natural-sounding audio.

You can generate speech for:

* The entire questionnaire
* Individual questions

This feature improves accessibility and supports audio-assisted interviewing.

> **For detailed information, refer to the Text-to-Speech documentation.**

#### Questionnaire Appearance & Navigation

Customize how the questionnaire is presented to respondents.

**Questionnaire Logo**

Upload a logo displayed in the questionnaire header.

Available logo settings include:

* Logo Width
* Logo Height
* Logo Fit

  * Contain
  * Cover
  * Fill

**Navigation Layout**

Choose how respondents move through the questionnaire.

Available layouts include:

* Original Layout
* Original Structure
* Show All Questions on One Page
* Show One Question Per Page
* Show One Input Field Per Page

**Start Page**: Enable a dedicated introduction page before respondents begin answering questions.

**Auto-Advance**: Automatically move respondents to the next page after completing the current one.

**Navigation Buttons**: Configure navigation controls, including:

* Show or hide navigation buttons
* Button alignment

  * Top
  * Bottom
  * Top and Bottom
* Show Previous Page button

**Progress Indicators**: Choose whether to display:

* Progress Bar
* Table of Contents (TOC)

These help respondents understand their progress through the questionnaire.

**Review Before Submit**: Allow respondents to review all answers before submitting the questionnaire.

**Button Labels**: Customize the text displayed on navigation buttons, including:

* Previous Page
* Next Page
* Complete Questionnaire

#### Default Question Settings

Configure default settings applied across all questionnaire questions.

Available settings include:

**Question Order**:
* Original
* Random

**Question Title Alignment**:
* Top
* Bottom
* Left

**Description Alignment** :
* Under the Question Title
* Under the Input Field

**Question Numbering**: Available numbering styles:

* Auto Numbering
* Reset on Each Page
* Recursive Numbering
* No Numbering

**Question Title Pattern**:

Choose how numbering and required indicators appear.

Examples:

* **1. Question Title***
* **1. *Question Title**
* **1. Question Title**

**Required Symbols**: Specify where required symbols appear.

Options:
* Top
* Bottom

### Additional Question Behavior
Configure options including:

* Focus first question on a new page
* Restrict answer length
* Restrict comment length
* Comment area height
* Automatically expand text areas
* Allow respondents to resize text areas

#### Pages

Manage questionnaire pages.

Available options include:

* Page Name
* Page Title
* Add New Page
* Show Page Titles
* Show Page Numbers
* Enable Grid Layout

These settings help organize long questionnaires into manageable sections.

#### Conditions

Configure questionnaire logic.

Available tools include:

**Custom Variables**: Create reusable variables for calculations and conditional logic.

**Triggers**: Automatically perform actions when specified conditions are met.

Triggers can be used to:

* Set values
* Skip questions
* Complete questionnaires
* Display messages

#### Data

Configure how questionnaire data is stored and updated. Available settings include:

**Clear Hidden Question Values**: Choose when hidden question values should be removed.

Options include:

* Never
* Upon Questionnaire Completion
* When the Question Becomes Hidden
* When the Question or Its Panel/Page Becomes Hidden

**Update Input Field Values**: Specify when entered values should be processed.

Options:

* On Lost Focus
* While Typing

**Auto-Save Progress**: Automatically save questionnaire progress whenever respondents move between pages.

**Save "Other" Values Separately**: Store values entered for **Other** answer options as separate data fields.

#### Validation

Control how questionnaire validation behaves.

Options include:

**Focus First Invalid Question**: Automatically move respondents to the first question containing validation errors.

**Run Validation**: Choose when validation occurs:

* When switching to the next page
* After an answer changes
* Upon questionnaire completion

#### Thank You Page

Configure what respondents see after submitting the questionnaire. Available options include:

* Redirect to an external URL
* Dynamic redirect URL
* Show or hide the Thank You page
* Customize Thank You page content
* Dynamic Thank You messages
* Loading message while the questionnaire initializes
* Message displayed if the respondent has already completed the questionnaire



#### Quiz Mode

Enable quiz functionality for assessments and knowledge tests. Available settings include:

**Timer**:

Enable a countdown timer for:

* The entire questionnaire
* Individual pages
* Both

**Timer Alignment**: 
Display the timer:

* Top
* Bottom

**Time Limits**:

Configure:

* Total questionnaire time limit
* Time limit per page

**Timer Mode**: Available modes include:

* Page
* Questionnaire
* Both

Quiz Mode is useful for certification exams, assessments, training programs, and timed evaluations.

## Summary

The **Designer** tab is the central workspace for creating and configuring questionnaires in VoxDash. It brings together all the tools needed to design survey content, organize questions, configure questionnaire behavior, and manage advanced features from a single interface.

Using the Designer tab, you can:

* Build questionnaires using a wide variety of question types.
* Organize questions with pages, sections, tags, and search filters.
* Configure questionnaire-wide settings such as appearance, navigation, validation, and data handling.
* Manage multilingual questionnaires and different data collection modes.
* Estimate the **Length of Interview (LOI)** to support project planning and budgeting.
* Generate **Text-to-Speech (TTS)** audio to improve accessibility and support audio-assisted surveys.
* Create and manage **Quota Groups** to control sample distribution during data collection.
* Define questionnaire logic using conditions, custom variables, and triggers.
* Customize the respondent experience with navigation settings, progress indicators, timers, and personalized Thank You pages.

The Designer tab serves as the foundation for questionnaire authoring, enabling you to create professional, scalable, and highly configurable surveys for data analytics, market research, and eCommerce applications. Before publishing your questionnaire, review all settings to ensure the survey behaves as expected and provides the best possible experience for respondents.
